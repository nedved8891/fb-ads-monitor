Privacy Policy

Last updated: April 2026

1. Introduction

FB Ads Monitor ("the Add-on") is a Google Sheets Add-on that monitors Facebook advertising campaigns. This Privacy Policy explains how we handle your data.

2. Data We Access

The Add-on accesses the following data:

  • Google Sheets data — to read configuration and write monitoring results
  • Facebook access tokens — stored only in your personal Google Sheets
  • Facebook ad performance data — retrieved via Facebook Marketing API
  • Telegram bot token and chat ID — stored only in your personal Google Sheets

3. How We Use Your Data

  • All data is stored exclusively in your personal Google Sheets
  • No data is transmitted to any third-party servers except Facebook API and Telegram API
  • The developer has no access to your tokens, ad account data, or spreadsheet contents
  • Data is used solely to provide the ad monitoring functionality

4. Data Storage

All configuration data including Facebook access tokens and Telegram credentials are stored in your own Google Sheets document. We do not store any of your data on external servers.

5. Third-Party Services

The Add-on communicates with:

  • Facebook Marketing API — to retrieve ad performance data and pause ads
  • Telegram API — to send monitoring notifications

These services have their own privacy policies which we encourage you to review.

6. Data Security

You are responsible for the security of your Google Sheets document and the credentials stored within it. We recommend not sharing your spreadsheet with unauthorized users.

7. Your Rights

You can delete all data at any time by removing the Add-on and deleting your Google Sheets document. Uninstalling the Add-on will revoke all permissions granted to it.
